Flickr8k English-Hindi data, with synthetic image features.

Dataset is combined from two sources:
https://www.kaggle.com/datasets/adityajn105/flickr8k
https://github.com/rathiankit03/ImageCaptionHindi
One mismatching image id has been omitted.
The Hindi side is machine translation output.

The synthetic features have been generated using a Marian-NMT
Transformer_base model trained from the Hindi Visual Genome.
